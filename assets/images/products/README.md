# Product Image Folder

Put product photos in this folder (`assets/images/products`).

Current page wiring uses these file names:

- `mcp.png`
- `wireless-av-alarm.png`
- `io module.png`
- `smoke.png`
- `node.png`
- `wireless loop expansion card.png`

Recommended format: transparent `PNG`, longest side `>= 1200px`.
