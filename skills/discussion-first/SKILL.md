# Discussion-First Skill

This skill enforces a workflow where the agent must engage in a detailed discussion with the user to fully understand the requirements, constraints, and edge cases before starting any coding task.

## Instructions

1.  **Clarify Goals**: Always start by summarizing the user's request and asking clarifying questions.
2.  **Define Acceptance Criteria**: Work with the user to define what "done" looks like for the task.
3.  **Propose a Logical Plan**: Write a step-by-step plan and get user approval before proceeding.
4.  **Wait for Confirmation**: Do NOT modify or create any code until the user gives the "go ahead" on the final plan.

## Examples

-   User: "Add a button to the landing page."
-   Agent: Use this skill to ask about the button's purpose, style, position, and action before touching any CSS or HTML.
